# To-Do-list
A python based simple To-Do list application ,which containing add, remove, update, view tasks.

# Features & Functionalities 
1. Add Task – Add new task with description. 
2. View Tasks – Display all tasks with status (Pending/Done). 
3. Update Task – Edit a task description. 
4. Mark Task as Done – Change task status from Pending → Done. 
5. Delete Task – Remove a task from the list. 
6. Exit Program – Close the application.
